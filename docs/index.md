## What this is

A reading aid I curate monthly. Not real-time. Not comprehensive. Just the stories I think are worth knowing about, organized so I can come back to a category and not lose the thread.

## Sourcing rules

- **Authoritative outlets first.** Reuters, AP, Bloomberg, FT, NYT, WSJ for general; Nature, Science, NEJM for science; MIT Tech Review, Ars Technica, The Verge, IEEE Spectrum for tech.
- **"Significant" = real-world impact, not hype.** Multi-outlet coverage is a positive signal.
- **No speculation as a headline story.** Opinion and analysis live inside the one-line "Why this matters."
- **Free links preferred** when a free alternative to a paywalled source exists.
- **Slim-month exception:** if a subcategory genuinely had fewer than 3 strong stories, the file ships 1–2 with a note. No padding.

This site is generated from a [GitHub repo](https://github.com/utexasdhong/News). To rebuild a month or add the next one, see the `meta/` folder in the repo for the design spec and implementation plan.
