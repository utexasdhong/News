# Products & Agents — April 2026

## 1. [Google Gemini Enterprise Agent Platform GA's at Cloud Next](https://cloud.google.com/blog/products/ai-machine-learning/introducing-gemini-enterprise-agent-platform)
**Source:** Google Cloud Blog · **Date:** 2026-04-22

Why this matters: Beyond the platform announcement, Google rolled out concrete agent products: Conversational Agents v2, Agent Garden with prebuilt vertical agents, and an Agent Dev Kit for custom builds. It's the first end-to-end agent stack from a hyperscaler that ships with enterprise governance hooks (audit logs, identity, data residency) on day one.

## 2. [GPT-5.5's OSWorld score reframes what "AI agent" means](https://medium.com/@sanjeevpatel3007/april-2026-ai-models-every-major-release-reviewed-6ea03d7bc0b7)
**Source:** Medium AI release roundup · **Date:** 2026-04-23

Why this matters: 78.7% on OSWorld-Verified — the benchmark that measures whether a model can actually operate a desktop OS, click buttons, navigate menus, fill forms — is roughly double where the field sat a year ago. Computer-use agents are now in the "useful for narrow workflows" zone, not the "interesting demo" zone.

## 3. [EY embeds agentic AI in audit at global scale](https://www.ey.com/en_gl/newsroom/2026/04/ey-launches-enterprise-scale-agentic-ai-to-redefine-the-audit-experience-for-the-ai-era)
**Source:** EY · **Date:** 2026-04-07

Why this matters: A multi-agent system inside EY Canvas running on Azure / Foundry / Fabric — used on real client engagements. It's the most visible production deployment of agentic AI in a regulated profession to date, and a template that PwC, Deloitte, and KPMG will be forced to match within the year.

## 4. [Writer ships autonomous, trigger-driven agents](https://venturebeat.com/technology/writer-launches-ai-agents-that-can-act-without-prompts-taking-on-amazon-microsoft-and-salesforce)
**Source:** VentureBeat · **Date:** 2026-04

Why this matters: Writer's new agents respond to data-source changes (a new ticket, a CRM update, a contract upload) rather than user prompts. That's the architectural shift the enterprise-agent category has been groping toward for two years — agents as ambient background workers, not as chatbots-with-tools.

## 5. [NVIDIA's open Agent Toolkit pairs with hyperscaler platforms](https://nvidianews.nvidia.com/news/ai-agents)
**Source:** NVIDIA Newsroom · **Date:** 2026-04

Why this matters: NVIDIA is positioning its Agent Toolkit — open-source models plus orchestration software — as the substrate underneath the Google / Microsoft / AWS agent platforms. It's the same play NVIDIA made in training (own the hardware + the CUDA layer); now it's running it at the agent layer.
